# DjangoOneBlog
基于Django的个人博客 开发与**2019.7.22**日  www.3io.cc

### 功能介绍

* 文章的发布，修改，删除（基于Django-Admin，集成了md编辑器）
* 文章可以按类型显示，可按文章内容及标题进行结果搜索
* 基于CBV实现（低阶）
* 侧边栏功能，最新文章，最多阅读，标签云等。
* 完整的评论功能，包括发表回复评论，以及评论的邮件提醒，支持`Markdown`。
* 基于Django2.2 , mysql
